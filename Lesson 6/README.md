## **Math Functions in Python**

1. `abs(x)`: The `abs()` function returns the absolute value of a number `x`. The absolute value is the positive value of `x`, irrespective of its sign. âœ–ï¸

```python

```

   The `abs()` function can be useful when you want to ensure that a value is positive.

2. `round(x)`: The `round()` function rounds a number `x` to the nearest integer. If the fractional part of `x` is less than 0.5, it rounds down; otherwise, it rounds up. ğŸ”„

```python

print(rounded_x)  # Output: 4
```

   The `round()` function is handy when you need to round off numbers to a specific precision.
